Proposal

The project I have in mind is a service that will allow people to search for a wide variety of Dungeons and Dragons spells.
While they search for spells that they want, they'll be able to favorite spells they like, as well as create a customized spell list
for any character they're using, which can be saved and edited at any time. Details such as spell levels and class specific spells
will be checked to ensure the spell list is appropriate.

This will be for frequent D&D players who want a simpler way to find and organize spells for their characters and campaigns.
The goal is to provide an easy method of searching for spells, and allow an immediate method to create a spell list for their use.

I will be using the open5e api and bulma for this project.

Major Features

  • Search for various D&D spell for use and reference
  • Favorite frequently used and viewed spells for quick and easy viewing
  • Create customizeable spell lists that can be used as a reference for character sheets
  • Save these spell lists for future reference, which can be viewed in favorites
